# Cloning an existing Environment
Starliner provides an easy mechanism to clone your environment, including all services and their corresponding configurations.
![New Environment](/img/cloning-environment/new-environment.png)
![Clone Environment](/img/cloning-environment/clone-environment.png)
Change the connected branch, so pushes to the connected branch will automatically be redeployed to the Staging environment.
![Change Connected Branch](/img/cloning-environment/connected-branch.png)
